# Features Overview: The Two-Speed UX

Lighthouse OS features a Two-Speed User Experience, catering directly to the professionals coordinating aid, while remaining accessible to the global public and advocacy groups.

## Level 1: Deep Analytics (For UN Professionals and Donor Advisors)
This suite of features is built specifically to support the rigorous demands of humanitarian coordination.

### 1. The Decision Impact Simulator (Prognostic Sandbox)
* **Function:** Users input hypothetical funding amounts.
* **Coordinator Value:** Provides proactive foresight. The system recalculates the global Overlooked Ranking instantly, showing how the allocation balances global inequity and estimating the Additional People Reached using historical cost-per-beneficiary data. It allows coordinators to test scenarios before committing funds.

### 2. Multi-Year Structural Timeline
* **Function:** Plots 5-year historical HRP targets versus actual disbursements.
* **Coordinator Value:** Distinct visual markers separate Sudden Onset Needs from Chronic Structural Neglect, allowing donor advisors to adjust their strategies based on the structural reality of the crisis.

### 3. Data Health and Transparency Matrix
* **Function:** Visually maps where data is missing or outdated.
* **Coordinator Value:** Enforces radical transparency. Highlights countries with active needs but no formal Humanitarian Response Plan, ensuring coordinators know exactly what blind spots exist in their operational planning.

### 4. Advanced MCP SQL Querying
* **Function:** Natural language to complex Databricks SQL execution.
* **Coordinator Value:** Reduces time-to-action. Analysts can ask complex conditional queries without knowing SQL syntax, instantly retrieving the specific cross-sections of data required for emergency meetings.

## Level 2: Accessible Impact (For the Public, Journalists, and Advocates)
This secondary interface translates complex UN data into understandable formats to drive global awareness and support.

### 1. The Narrative Dashboard
* Translates raw Coverage Ratios into human-centric metrics to help the public understand the scale of the funding gap.

### 2. AI Co-Pilot
* The AI agent acts as a humanitarian tutor, explaining complex funding mechanisms (CERF, CBPF, HRPs) to users trying to understand the UN ecosystem.

### 3. Shareable Insight Cards
* One-click generation of data-accurate infographics for social media awareness and journalism, helping grassroots organizations advocate for underfunded crises using verified UN data.
